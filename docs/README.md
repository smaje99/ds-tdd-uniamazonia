# Data Structures Study Guide

This folder complements the code in `src/main/java` with short study notes for
each topic covered in the repository.

Each document is written for students who need to:

- understand the invariant of the structure before coding
- predict the expected complexity of each operation
- identify the difference between the `scratch`, `collections`, and `linear`
  approaches
- avoid the most common mistakes that make contract tests fail

## How To Use These Notes

1. Read the invariant first.
2. Compare the expected complexity with the implementation you are editing.
3. Review the pitfalls section before running tests.
4. Use the notes together with the Javadoc in `src/main/java`.
